Dev Template
============

# Get it!

```sh
git clone https://github.com/hontas/dev-tmpl.git
```

# Use it!

Rename the folder from `dev-tmpl` to that of your choosing and cd into it.

```sh
npm install
node setup
```

# Test

```sh
npm install -g mocha
npm test
```
