Dev Template
============

# Install

```sh
git clone https://github.com/hontas/dev-template.git
npm install
bower install
```

# Setup

Rename the folder from `dev-template` to that of your choosing.

```sh
node setup
```

# Test

```sh
npm install -g mocha
npm test
```
